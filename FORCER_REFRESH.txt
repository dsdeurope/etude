🔥 MÉTHODE 1: Hard Refresh (RAPIDE)

Dans le preview ou votre navigateur:

Windows/Linux:  Ctrl + Shift + R
Mac:            Cmd + Shift + R

OU

Windows/Linux:  Ctrl + F5
Mac:            Cmd + Shift + Delete (puis videz le cache)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔥 MÉTHODE 2: Vider le cache complètement
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Dans Chrome/Edge:
1. Ouvrez http://localhost:3000
2. Appuyez sur F12 (ouvre les DevTools)
3. Faites un clic droit sur le bouton de rafraîchissement
4. Sélectionnez "Vider le cache et actualiser de force"

Dans Firefox:
1. Ouvrez http://localhost:3000
2. Ctrl+Shift+Delete
3. Cochez "Cache"
4. Cliquez "Effacer maintenant"
5. Rafraîchissez la page (F5)

1. Ouvrez une fenêtre de navigation privée/incognito:
   - Chrome/Edge: Ctrl+Shift+N
   - Firefox: Ctrl+Shift+P
   - Safari: Cmd+Shift+N

2. Dans cette fenêtre, allez sur: http://localhost:3000

3. ✅ Les LED seront visibles car pas de cache!
