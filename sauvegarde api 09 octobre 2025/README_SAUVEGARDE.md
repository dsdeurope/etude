## 🚀 Démarrage rapide

### 1. Restaurer les fichiers
```bash
# Copier le backend
cp -r backend/* /app/backend/

# Copier le frontend  
cp -r frontend/* /app/frontend/
```

### 2. Installer les dépendances
```bash
# Backend
cd /app/backend
pip install -r requirements.txt

# Frontend
cd /app/frontend
yarn install
```

### 3. Redémarrer les services
```bash
sudo supervisorctl restart all
```
